# Credit Card Fraud Detection & Risk Analysis

## Project Overview
This project analyzes 280,000+ credit card transaction records to identify fraudulent activity patterns, validate fraud-related KPIs, and visualize risk trends using Excel, SQL, Python, and Power BI.

## Tools Used
- Excel: Data cleaning, organization, pivot tables, and dashboard creation
- SQL: Fraud queries and KPI extraction
- Python: Data validation and metric verification
- Power BI: Interactive dashboard visualization

## Business Questions
- How many transactions were fraudulent?
- What was the total fraud loss?
- What was the average fraud transaction amount?
- Which transaction had the highest fraud amount?
- What hours showed the most fraud activity?

## Project Workflow
1. Cleaned and organized the transaction dataset in Excel.
2. Created pivot tables and Excel dashboard visuals.
3. Wrote SQL queries to extract fraud-related KPIs.
4. Used Python to validate key metrics and confirm data accuracy.
5. Built a Power BI dashboard to visualize fraud trends and risk patterns.
